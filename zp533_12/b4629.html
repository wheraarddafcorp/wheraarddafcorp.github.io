説明: Overview
ESP32-S3 Microcontroller, 2.4 GHz Wi-Fi Development Board, dual-core processor with frequency up to 240 MHz, Supports development environments such as ESP-IDF, Arduino, MicroPython,etc

Key Features
1. Adopts ESP32-S3R2 chip with Xtensa 32-bit LX7 dual-core processor, capable of running at 240 MHz
2. Integrated 512KB SRAM, 384KB ROM, 2MB PSRAM, 16MB Flash memory
3. Integrated 2.4GHz Wi-Fi and Bluetooth LE dual-mode wireless communication, with superior RF performance
4. Type-C connector, easier to use
5. Onboard CH343 and CH334 chips can meet the needs of USB and UART development via a Type-C interface
6. Onboard DC-DC chip MP28164, high efficiency DC-DC buck-boost chip with load current up to 2A
7. 27 × multi-function GPIO pins
8. Rich peripheral interfaces, including full-speed USB OTG, SPI, I2C, UART, ADC, PWM and DVP (8-bit ~ 16-bit camera interface), LCD interfaces (8-bit ~ 16-bit parallel RGB, I8080, MOTO6800), etc. to achieve various functions flexibly
9. Castellated module allows soldering directly to carrier boards
10. Supports multiple low-power operating states, adjustable balance between communication distance, data rate and power consumption to meet the power requirements of various application scenarios
11. Supports Raspberry Pi Pico expansion board ecology, suitable for artificial intelligence IoT application development
