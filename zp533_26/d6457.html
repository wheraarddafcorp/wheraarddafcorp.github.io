タックルキャリアー 4025＋ショルダーベルト
Shoulder Belt Style
長距離移動が快適なように、斜めかけ対応ロングサイズの
ショルダーベルトを装備。
バックルで脱着可能なのでベルトを外して、バッカン丸洗いOK。

※ロッドを立てた状態で持ち上げる時は、
先にハンドルで持ち上げてからショルダーベルトを肩に掛けて下さい。
ショルダーベルトだけで持ち上げますと、バランスを崩し倒れて、
ロッドを破損します。

Rod Stand Style
3カ所のロッドスタンドを装備。
左右の2ヶは、31φまでを収納でき、
8ft前後のライトタックルゲームロッドやエギングロッドを
立てた状態で収納できます。
センタースタンドは45φまで対応できるので
小継ネットや小継ギャフも立てた状態で収納できます。
ロッドを地面置きしないのでロッドやリールをキズ付けずに、
タックルチェンジが素早く行えます。

Bakkan Style
サビに強いステンレス製フレームや半開固定ボタンを採用した
EVAセミハードバッカンは、タックル収納からキープバケツ等、
状況にあわせた使い方ができます。

Offset Handle Style
ロッドをセットした時の重量バランスを考え、
ハンドルはロッドスタンド側に28mmオフセットしています。
またハンドル内側の凸凹を合わせると、
すぐに握れる合体ハンドルになります。

Assist Arm Style
バッカンが倒れずロッドを安定して立てられるように
折りたたみ可能なアシストアームを装備。
使用時はアシストアームを倒しロックリングで完全固定して下さい。

SPEC
サイズ 400×250×270mm
重量(カタログ値) 1700g
ロッドスタンド径 φ31mm×2、φ45mm×1
ショルダーベルト最大サイズ 180cm
